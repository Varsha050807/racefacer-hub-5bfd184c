# Racefacer Frontend

## Project Overview
Racefacer is a racing management platform dashboard supporting three user roles:
- Admin
- Track Owner
- Racer

This is a role-based dashboard system built using modern frontend technologies.

---

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn UI

---

## Features
- Role-based dashboards
- Modern dark themed UI
- Responsive layout
- Sidebar navigation based on user role
- Dashboard analytics and data panels (UI simulation)

---

## Setup Instructions

### Install Dependencies
npm install

### Run Development Server
npm run dev
